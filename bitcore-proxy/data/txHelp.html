What follows is an exhaustive description of the different types of 'tx' requests you can pose to this server:

Get Transactions by blockHeight

GET `/api/BCH/mainnet/tx?blockHeight=12` (example)

Get Transactions by blockHash

GET `/api/BCH/mainnet/tx?blockHash=:blockhash` (example)

Get Transaction by txid

GET `/api/BCH/mainnet/tx/:txid` (example)

Get Authhead

GET `/BCH/mainnet/tx/:txid/authhead` (unsupported)

Get detailed data about a transaction, including inputs and outputs

GET `/BCH/mainnet/tx/:txid/coins` (unspent coinbase, example2)
