What follows is an exhaustive description of the different types of 'address' requests you can pose to this server:

Get UTXOs

GET /address/utxo/{address}
Address utxos single. Example

POST /address/utxo
Address utxo bulk.

Request:
{
  "addresses": [
    "bitcoincash:qzs02v05l7qs5s24srqju498qu55dwuj0cx5ehjm2c",
    "bitcoincash:qrehqueqhw629p6e57994436w730t4rzasnly00ht0"
  ]
}

Get Details

GET /address/details/{address}
Returns the details of an address including balance. Example

POST /address/details
Returns the details of one or more addresses including balance.

Request:
{
  "addresses": [
    "bitcoincash:qzs02v05l7qs5s24srqju498qu55dwuj0cx5ehjm2c",
    "bitcoincash:qrehqueqhw629p6e57994436w730t4rzasnly00ht0"
  ]
}

GET /address/unconfirmed/{address}
Unconfirmed transactions for an address. Example

POST /address/unconfirmed
Unconfirmed transactions for array of addresses.

Request:
{
  "addresses": [
    "bitcoincash:qzs02v05l7qs5s24srqju498qu55dwuj0cx5ehjm2c",
    "bitcoincash:qrehqueqhw629p6e57994436w730t4rzasnly00ht0"
  ]
}

GET /address/transactions/{address}
Transactions for an address. Example

POST /address/transactions
Transactions for an array of addresses.

Request:
{
  "addresses": [
    "bitcoincash:qzs02v05l7qs5s24srqju498qu55dwuj0cx5ehjm2c",
    "bitcoincash:qrehqueqhw629p6e57994436w730t4rzasnly00ht0"
  ]
}
